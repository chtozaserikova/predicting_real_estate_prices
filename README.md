# Цель проекта
Создать модель, прогнозирующую цену на 1-комнатную квартиру в Москве по некоторым параметрам (см. cian.csv) для поиска наиболее выгодного варианта. Чем выше прогнозируемая цена в сравнении с указанной на сайте, тем выгоднее предложение.
Единственная задача, которую я пытаюсь решить - облегчить себе поиск жилья.

# Требования
1. ChromeDriver версии 104 (и, соответственно, аналогичная бета-версия Хрома) для работы с selenium. При работе с ChromeDriver 103 вылетает ошибка, связанная с багом самого хрома;
2. Названия классов на сайте циан (а именно последние 4 символа в названии) часто обновляются, поэтому в случае ошибки при клонировании репозитория в первую очередь стоит проверить сайт;
3. !pip install -r req.txt после создания виртуальной среды
